# 漢字化エスペラント → アルファベット変換アプリ

## 概要
このStreamlitアプリは、漢字で表現されたエスペラント語根を元のアルファベット表記に戻すツールです。

## 機能
- ✅ `PEJVO/PIV 2026-06-20` の新しい漢字割当案に対応
- ✅ 旧スニペット最小CSVとのモード切替
- ✅ カスタムCSVファイルのアップロード対応
- ✅ 漢字とアルファベットが混在した文章に対応
- ✅ 上付き識別子（例: `藻ᶠᶜ`, `此ᶜ̂`）を含む漢字表記に対応
- ✅ 変換結果はすべて小文字で出力
- ✅ テキストファイルとしてダウンロード可能

## インストール

### 1. 必要なパッケージのインストール
```bash
pip install streamlit pandas
```

### 2. ファイルの配置
以下のファイルを同じディレクトリに配置してください：
- `esperanto_converter.py` - Streamlitアプリケーション
- `data/pejvo_piv_20260620_reverter.csv` - 新しい標準対応表
- `エスペラント語根-漢字対応表_スニペット用最小限.csv` - 旧スニペット最小対応表

## 実行方法

### ローカル環境での実行
```bash
streamlit run esperanto_converter.py
```

ブラウザで自動的に開かない場合は、以下のURLにアクセスしてください：
```
http://localhost:8501
```

## 使い方

### 1. 漢字割当案の選択
- **PEJVO/PIV 2026-06-20**: 新しい標準割当案（推奨）
- **旧スニペット最小CSV**: 以前の最小対応表
- **カスタムCSV**: 独自の対応表をアップロード

### 2. 漢字化エスペラント文の入力
左側のテキストエリアに漢字化されたエスペラント文を入力します。

**入力例：**
```
何时 西o 遇as 东方on 和 上置as 东方an 服on, 一 独a 语o 获as 二 观ojn.
```

### 3. 変換の実行
「🔄 変換する」ボタンをクリックします。

### 4. 結果の確認
右側のテキストエリアに変換されたアルファベットのエスペラント文が表示されます。

**出力例：**
```
mi aŭdis, ke inter kontraŭpli ol semajno kaj de du multaj homoj inter kontraŭsamaj lokoj, ke kumino tre bone konvenas por la ŝafaĵa grandplado "jingisukan".
```

### 5. ダウンロード（オプション）
「📥 テキストをダウンロード」ボタンをクリックして、変換結果をテキストファイルとして保存できます。

## CSVファイル形式

カスタムCSVファイルを使用する場合は、以下の形式に従ってください：

```csv
esperanto,kanji,priority
bon,良,0
fic,藻ᶠᶜ,0
bangi,藻ᴮ,0
```

### 形式の詳細
- **列1**: エスペラント語根（アルファベット）
- **列2**: 対応する漢字（上付き識別子を含む場合あり）
- **列3**: 漢字化優先順位（任意。小さいほど優先）
- **ヘッダー行**: あってもなくても可
- **エンコーディング**: UTF-8

## 新しい割当案CSVの生成

`kanji-esperanto-monaco` 側の `all.json` から reverter 用CSVを再生成できます：

```bash
python3 tools/build_reverter_dictionary_from_all_json.py ../kanji-esperanto-monaco/all.json data/pejvo_piv_20260620_reverter.csv
```

生成時に、`c^i-tie` や `u^` などの `^` 表記は `ĉi-tie`、`ŭ` などの小文字エスペラント表記へ戻します。

### 特殊文字について
同じ漢字が複数の語根に対応する場合、上付き識別子で区別します：

```csv
alg,藻
fic,藻ᶠᶜ
bangi,藻ᴮ
```

## 動作原理

1. 選択した割当案から漢字→エスペラント語根の辞書を作成
2. 入力文を1文字ずつ走査
3. 上付き識別子を含む漢字表記を、長いものから優先して語根へ変換
4. 同じ長さの候補は `priority` の小さいものを優先
5. 変換されない文字、アルファベット、数字、記号はそのまま保持
6. 最終的にすべて小文字に変換

## トラブルシューティング

### 標準CSVファイルが見つからない
以下のパスにCSVファイルを配置してください：
```
data/pejvo_piv_20260620_reverter.csv
```

または、サイドバーで「カスタム CSV」を選択して、独自のCSVファイルをアップロードしてください。

### 変換がうまくいかない
- CSVファイルの形式を確認してください
- 漢字と特殊文字の組み合わせが正しいか確認してください
- 入力文が正しい形式か確認してください
