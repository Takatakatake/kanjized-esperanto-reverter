# 漢字化エスペラント → アルファベット変換アプリ

## 概要
このStreamlitアプリは、漢字で表現されたエスペラント語根を元のアルファベット表記に戻すツールです。

## 機能
- ✅ デフォルトの対応表を使用
- ✅ カスタムCSVファイルのアップロード対応
- ✅ 漢字とアルファベットが混在した文章に対応
- ✅ 特殊文字（ʜ, ɪ, ʟ, ᴠなど）も正しく処理
- ✅ 変換結果はすべて小文字で出力
- ✅ テキストファイルとしてダウンロード可能

## インストール

### 1. 必要なパッケージのインストール
```bash
pip install streamlit pandas
```

### 2. ファイルの配置
以下のファイルを同じディレクトリに配置してください：
- `esperanto_converter.py` - Streamlitアプリケーション
- `エスペラント語根-漢字対応表_スニペット用最小限.csv` - デフォルトの対応表（オプション）

## 実行方法

### ローカル環境での実行
```bash
streamlit run esperanto_converter.py
```

ブラウザで自動的に開かない場合は、以下のURLにアクセスしてください：
```
http://localhost:8501
```

## 使い方

### 1. CSVファイルの設定
- **デフォルトモード**: そのまま使用（推奨）
- **カスタムモード**: サイドバーで「カスタムCSVファイルを使用」をチェックし、独自のCSVファイルをアップロード

### 2. 漢字化エスペラント文の入力
左側のテキストエリアに漢字化されたエスペラント文を入力します。

**入力例：**
```
我 听is, ke 间 反更 比 周o 和 从 二 多样aj 人ʜoj 内 反同aj 处ʟoj, ke kumino 很 好e 适as 为 la 羊物a 大盘o "jingisukan".
```

### 3. 変換の実行
「🔄 変換する」ボタンをクリックします。

### 4. 結果の確認
右側のテキストエリアに変換されたアルファベットのエスペラント文が表示されます。

**出力例：**
```
mi aŭdis, ke inter kontraŭpli ol semajno kaj de du multaj homoj inter kontraŭsamaj lokoj, ke kumino tre bone konvenas por la ŝafaĵa grandplado "jingisukan".
```

### 5. ダウンロード（オプション）
「📥 テキストをダウンロード」ボタンをクリックして、変換結果をテキストファイルとして保存できます。

## CSVファイル形式

カスタムCSVファイルを使用する場合は、以下の形式に従ってください：

```csv
エスペラント語根,漢字
mi,我
aŭd,听
kun,共
inter,内
```

### 形式の詳細
- **列1**: エスペラント語根（アルファベット）
- **列2**: 対応する漢字（特殊文字を含む場合あり）
- **ヘッダー行**: 不要
- **エンコーディング**: UTF-8

### 特殊文字について
同じ漢字が複数の語根に対応する場合、特殊文字で区別します：

```csv
oni,人ᴏ
hom,人ʜ
lok,処ʟ
viand,肉ᴠ
```

## 動作原理

1. CSVファイルから漢字→エスペラント語根の辞書を作成
2. 入力文を1文字ずつ走査
3. 漢字（+特殊文字）を検出したら辞書から対応する語根に変換
4. アルファベット、数字、記号はそのまま保持
5. 最終的にすべて小文字に変換

## トラブルシューティング

### デフォルトCSVファイルが見つからない
以下のパスにCSVファイルを配置してください：
```
/mnt/user-data/uploads/エスペラント語根-漢字対応表_スニペット用最小限.csv
```

または、サイドバーで「カスタムCSVファイルを使用」をチェックして、独自のCSVファイルをアップロードしてください。

### 変換がうまくいかない
- CSVファイルの形式を確認してください
- 漢字と特殊文字の組み合わせが正しいか確認してください
- 入力文が正しい形式か確認してください
