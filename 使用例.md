# 漢字化エスペラント変換 - 使用例

## PEJVO/PIV 2026-06-20

### 基本例

入力:

```text
何时 西o 遇as 东on 和ᴷ 上置as 东an 衣on, 一 唯a 语o 获as 二 显ojn — 两 美ajn —, 和ᴷ 生成as 新a 解o.
```

出力:

```text
kiam okcidento renkontas orienton kaj surmetas orientan veston, unu sola lingvo akiras du aspektojn — ambaŭ belajn —, kaj naskiĝas nova kompreno.
```

### 上付き識別子つき表記

入力:

```text
藻ᶠᶜoj 和ᴷ 藻ᴮoj
```

出力:

```text
ficoj kaj bangioj
```

`藻` だけなら `alg`、`藻ᶠᶜ` なら `fic`、`藻ᴮ` なら `bangi` のように、上付き識別子を含めて判定します。`和`（単体）は `pac`、`和ᴷ` が `kaj` なので、`kaj` を出すには `和ᴷ` を使います。

### 最新割当の識別子つき表記

入力:

```text
此 和ᴷ 此ᶜ
```

出力:

```text
ĉi kaj cis
```

## 旧スニペット最小CSV

旧モードを選ぶと、リポジトリに元から入っていた対応表で変換できます。

入力:

```text
我 听is 人ʜoj
```

出力:

```text
mi aŭdis homoj
```

## 変換の特徴

- 漢字表記が長いものを先に変換します。
- 上付き識別子も漢字側の表記長に含めます。
- 同じ長さの重複キーでは、priority が小さいものを優先します。
- アルファベット部分や未知の文字は変換せずに残します。
- 出力はすべて小文字になります（残した文字も含め最終的に小文字化するため、入力の大文字は保持されません）。
