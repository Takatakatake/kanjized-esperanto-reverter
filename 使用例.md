# 漢字化エスペラント変換 - 使用例

## 変換例

### 例1: 基本的な変換

**入力（漢字化エスペラント）:**
```
我 听is, ke kumino 很 好e 适as 为 la 羊物a 大盘o.
```

**出力（アルファベット）:**
```
mi aŭdis, ke kumino tre bone taŭgas por la ŝafaĵa plado.
```

**日本語訳:**
「私は聞きました、クミンが羊肉の大皿料理にとても良く合うことを。」

---

### 例2: 完全な文章

**入力（漢字化エスペラント）:**
```
我 听is, ke 间 反更 比 周o 和 从 二 多样aj 人ʜoj 内 反同aj 处ʟoj, ke kumino 很 好e 适as 为 la 羊物a 大盤o "jingisukan".
我 实际e 试is kuminon 共 jingisukan, 和 我 全e 同意as: kumino 真e 好e 协议as kun la 羊物a 大盘o.
从 今, 何时 我 吃os jingisukan, 我 决is 全时 辛i la 肉ᴠon 以 kumino.
```

**出力（アルファベット）:**
```
mi aŭdis, ke dum malpli ol semajno kaj de du diversaj homoj en malsamaj lokoj, ke kumino tre bone taŭgas por la ŝafaĵa plado "jingisukan".
mi efektive provis kuminon kun jingisukan, kaj mi ĉie konsentas: kumino vere bone akordas kun la ŝafaĵa plado.
de nun, kiam mi manĝos jingisukan, mi decidis ĉiam spici la viandon per kumino.
```

**日本語訳:**
「私は聞きました、一週間もしないうちに二つの異なる場所の様々な人々から、クミンがジンギスカンという羊肉の大皿料理にとても良く合うことを。
私は実際にクミンをジンギスカンと一緒に試してみました、そして私は完全に同意します：クミンは本当に良く羊肉の大皿料理と調和します。
今から、私がジンギスカンを食べる時はいつでも、私はいつもクミンで肉を味付けすることに決めました。」

---

## 変換の特徴

### 1. 漢字の変換
漢字はエスペラント語根に変換されます：
- 我 → mi (私)
- 听 → aŭd (聞く)
- 很 → tre (とても)
- 好 → bon (良い)

### 2. 特殊文字の処理
同じ漢字が複数の語根に対応する場合、特殊文字で区別されます：
- 人ʜ → hom (人間)
- 人ᴏ → oni (一般の人々)
- 処ʟ → lok (場所)
- 肉ᴠ → viand (肉)

### 3. アルファベット部分の保持
既にアルファベットで書かれている部分はそのまま保持されます：
- is, as, os → 動詞の時制語尾
- aj, oj, on → 名詞・形容詞の格語尾
- kumino, jingisukan → 外来語や固有名詞

### 4. 小文字への変換
出力はすべて小文字に変換されます（エスペラントの慣習に従う）。

## よくある変換パターン

| 漢字化 | アルファベット | 意味 |
|--------|----------------|------|
| 我 | mi | 私 |
| 你 | vi | あなた |
| 他 | li | 彼 |
| 她 | ŝi | 彼女 |
| 和 | kaj | そして |
| 是 | est | である |
| 有 | hav | 持つ |
| 从 | de | から |
| 为 | por | ために |
| 共 | kun | と共に |
| 内 | en | 中に |
| 以 | per | によって |

## トラブルシューティング

### 変換されない漢字がある場合
- CSVファイルに該当する漢字のエントリがあるか確認してください
- 特殊文字が正しく入力されているか確認してください（例：人ʜ）

### 出力が期待と異なる場合
- 入力文の漢字とアルファベットの区切りが正しいか確認してください
- 空白や記号が正しく配置されているか確認してください
